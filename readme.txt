payware Mobile payments for WooCommerce

== Description ==
payware is B2B innovative multi-sided payment initiation and information exchange platform for e-commerce, merchants and financial institutions.
The plugin allows WooCommerce based shops to receive payments outside card networks from any payware enabled mobile payment application.
Supported payment initiation methods are QR code and text.
Transactions are processed through payware payment gateway. 

== Prerequisites ==
WooCommerce

== Installation ==

1. Upload `payware-payment-gateway.zip` to your WordPress deployment plugin directory. By default it is `/wp-content/plugins/` directory
2. Unpack the archive
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Enable the payment method on the page WooCommerce > Settings > Payments
5. Click on the 'Set Up' button for `Mobile payments by payware`
6. Fill in the plugin data
7. Use
